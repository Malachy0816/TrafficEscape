Project Description:
Traffic escape is a simple android and desktop app where the player must avoid oncoming traffic. Cars continuously move toward the player, and the player must move left or right to dodge them. The objective is to survive as long as possible without crashing. This project was created for my cross-platform application development module using .NET MAUI and C#

Technologies used: C#, .NET MAUI, XAML, Visual Studio

Setup Instructions:
1. Clone the repository.
2. Open project in visual studio 2022 and make sure .NET MAUI framework is installed.
3. Run project on preferred platform (Windows, android emulator).

Usage:
1. Select start game in the menu.
2. Click left or right to move the player to avoid oncoming cars.
3. collect power ups and coins throughout the game.
4. Avoid collisions for as long as possible
